#### **플로우**
- 온보딩 화면
	- 언어 선택화면
		- 기본 한국어 선택되어 있음
		- vpn으로 지역을 변경하여도 한국어로 고정
	- 목표 등급 선택
- AI 생성 콘텐츠 알림
- 로그인 화면
- 회원가입 화면
	- 회원가입 시 상단 되돌아가기 버튼 없음
	- 로그인 버튼이 회원가입 버튼과 겹치는 문제
	- 각 항목별 요구사항 미충족시 에러출력이 없음
	- 회원가입시 비밀번호 항목에 출력되는 비밀번호 조건과 가입 시도시 출력되는 비밀번호 조건이 다름
		- 조건이 미충족 되었음에도 다른 에러와 다르게 에러포커싱이 되지 않음
	- 회원가입 진행시 이메일이 전송중인 상태임에도 회원가입 완료 화면으로 이행됨.
- 비밀번호 재설정 화면
	- 가입되지 않은 이메일로 재설정 링크 송출시 안내가 없음
		- 해당 이메일로 가입되어있는 경우 송출된다는 안내가 필요할 듯
	- 재설정 링크 송출에 쿨타임이 존재하지 않음(연속 송출 가능)
- 읽기문제
	- 북마크/오프라인 문제
		- 이미 오프라인 저장한 문제라고 하더라도 오프라인 저장 버튼이 활성화 되어있다.
			- 북마크한 문제에서는 그러한 문제가 발생하지 않음
		- 문제를 북마크한 후, 저장된 학습 데이터 -> 문제 목록(북마크 해둔 문제를 풀어보세요)에 가도 문제는 존재하지 않는다.
		- 여기에 저장되는 것은 북마크한 문제가 아니라 오프라인 저장한 문제
		- !! 홈 화면에 북마크 문제로 이동하는 섹션이 별도로 존재
		- 북마크한 내용은 이곳에 저장된다.
		- 나의 북마크에서 
			- 단어 리스트 -> 북마크 해제시 상태 즉각 반영
			- 다만, 뒤로가기 버튼을 통해 홈 화면으로 복귀시 상태 미반영
			- 문제 복습 -> 문제 선택 -> 북마크 해제 -> 문제 목록 복귀시 상태 미반영
				- 다만 새로고침하면 정상적으로 동작
				- 마찬가지로 홈화면으로 가도 상태 미반영
				- 이런 상태 즉각 미반영 되는 사항은 오답문제 복습하기에서도 나타남..
