
## 회원가입 오류텍스트 문제

- 다른 항목과 다르게 비밀번호 확인은 판정이 늦다
	- 입력 즉시 에러/정상 판정되는 것이 아니라 다른 곳을 터치하는 등의 조작이 필요함.
- 비밀번호의 출력 문구 문제
	- 기본 포커스 출력 문구는 `8자 이상 20자 이하, 영문 대소문자가 포함되어야 합니다`
	- 그러나 8자 이하일 때의 오류 출력 문구는 `비밀번는 8자 이상이어야 합니다.`
	- 문구를 통일 시키는 것이 좋아 보인다.

## 저장된 학습 데이터 문구 혼동 문제

- 학습 페이지에 명시된 문구는 변경 확인

	- 진입 버튼과 진입한 페이지의 이름 불일치
		- 진입 버튼 : 오프라인 저장 데이터
		- 진입 페이지 : 저장된 학습 데이터
