## 개요

모의고사 리스트 페이지를 다음과 같이 분리하고자 한다.

- 허브 페이지
- 리스트 페이지
- 상세 페이지

## 요구 사항 

여기에 추가로 다음과 같은 컴포넌트가 필요하다

- nested router nav 컴포넌트

각 화면 별로 다음과 같은 데이터가 필요하다.

- 허브 페이지
	- 최근 학습 이력
	- 다가오는 시험?
- 리스트 페이지
	- 모의고사 데이터
- 상세 페이지
	- 모의고사 상세 데이터(by id)

## 허브 페이지 구성 요소

- 최근 학습 이력
- 영역별 학습
	- 듣기
	- 읽기
	- 등...
- 다가오는 시험?
- 혹은 간단한 현재 유저 정보를 기반으로 한 데이터셋
- 추천 모의고사(현재 불가능)

## 모의고사 리스트 페이지

- 다만 하단 구매/시험 등의 버튼은 변경이 필요
